# Hongmou Hu's Academic Homepage

This repository contains the source code for my personal academic website, hosted at [cleverhommer.github.io](https://cleverhommer.github.io/).

## About This Site

The site is built as a lightweight, single-page application that renders Markdown content dynamically. It showcases my research in Information Security, academic projects, and honors at the University of Science and Technology of China (USTC).

## Structure

- `index.html`: The main entry point and UI layout.
- `contents/`: Contains Markdown files (`home.md`, `publications.md`, `awards.md`) and configuration.
- `static/`: Assets including CSS, JavaScript, and images (photos/backgrounds).
